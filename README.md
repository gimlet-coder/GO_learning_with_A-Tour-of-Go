## 気がついた注意事項

学習していくうえで気になった注意点を書き記していきます

### 1. 同一フォルダ内における main 関数は 1 つのみ

* フォルダの中に main 関数が入ったファイルが複数あるとエラーがでます
    * 対策1: 関数を main から main_1 といったように変更する
        * main 関数が入っていない単体ファイルは実行できないためやや不自由
    * 対策2: フォルダをその都度作成して 1 フォルダ 1 main 関数の原則を守る (こちらを採用)
        * フォルダを毎回作る煩わしさはあるもののそのまま実行できる利便さを優先した
